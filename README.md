## Kurunt

Kurunt is a scalable message processing framework for real-time data. 

**Under Dev**

### Installation

```js
npm install -g kurunt
```

### Use

To launch Kurunt, from your terminal:

```
kurunt
```

Then to administer can open your browser at:

```
http://localhost:8888
```
