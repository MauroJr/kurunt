Streams Report

Install socket.io

You need to install socket.io to view the stream report. It's easy using npm from your terminal:

Firstly stop kurunt.

Ctrl + c

Now you can install socket.io into kurunt.

cd && npm install socket.io

Restart kurunt.

kurunt

Click the Report link.
